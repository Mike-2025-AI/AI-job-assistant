# AI 智能体实现指南 v2.0

> 求职助手 · Coze Bot 配置与接入文档

---

## 1. 概述

本项目的 AI 分析能力由 **Coze（扣子）平台** 提供，无需自建大模型，直接通过 Coze Bot 调用大语言模型完成简历分析。

**优势**：无需自己部署模型 · 配置提示词即可定义分析行为 · 可视化调试 · 支持多平台发布

---

## 2. Coze Bot 创建步骤

### 2.1 前提条件
- Coze 账号：https://www.coze.cn（国内版）

### 2.2 创建 Bot
1. Coze 控制台 → **"Bot"** → **"创建 Bot"**
2. 填写：Bot 名称`简历分析助手`、描述`帮助用户分析简历与岗位匹配度`
3. 点击 **"确认创建"**，复制页面中的 **Bot ID**（用于前端代码配置）

### 2.3 系统提示词
在 Bot 编辑页 → **"技能"** → **"大模型"** → 填入以下内容：

```
你是一位专业的简历评估专家。请根据以下简历和岗位信息，从六个维度进行深入分析：

## 评分维度（每项0-100分）
1. 技能匹配度  2. 经验匹配度  3. 学历匹配度
4. 行业匹配度  5. 薪资匹配度  6. 整体竞争力

## 输出格式（严格按此格式输出）
## 综合匹配度：X分
## 评级：[优秀/良好/一般/较差]

## 六维评分
技能匹配：X分 / 经验匹配：X分 / 学历匹配：X分
行业匹配：X分 / 薪资匹配：X分 / 整体竞争力：X分

## 匹配亮点
• ...

## 差距分析
• ...

## 改进建议
• ...

## 面试题推荐
• ...
```

### 2.4 模型选择
推荐使用 `doubao-pro-32k`（字节跳动大模型，性价比高）。

---

## 3. API Token 获取

1. Coze 控制台 → 右上角头像 → **"个人设置"** → **"开发者"** → **"API Token"**
2. 点击 **"创建 Token"**，填写名称后复制 Token（以 `pat_` 开头）
3. 在 Token 详情页添加 Bot 授权（将该 Token 绑定到"简历分析助手"Bot）

---

## 4. 前端配置

在 `analyze.html` 中填入两个值：

```javascript
const COZE_API_TOKEN = "pat_你的Token";
const BOT_ID = "你的BotID";
```

---

## 5. 调试方法

| 调试入口 | 说明 |
|----------|------|
| Coze 预览面板 | 模拟输入简历+岗位，实时查看 AI 输出格式 |
| 浏览器 Network 面板 | 查看 `/v3/chat` 请求是否成功，错误码对应含义 |

**常见错误码**：

| 状态码 | 含义 | 解决方案 |
|--------|------|----------|
| 401 | Token 无效/过期 | 重新获取 PAT |
| 403 | Token 未授权该 Bot | 在 Coze 控制台绑定授权 |
| 429 | 超过调用频率 | 等待 1 分钟后再试 |

---

## 6. 提示词优化建议

- **定期检查**：在 Coze 控制台查看调用日志，观察输出格式是否稳定
- **强化格式**：如 AI 输出格式有偏差，在提示词开头加"严格按照以下格式"
- **示例引导**：在提示词末尾加入一个完整的 JSON 示例，可显著提高格式一致性

---

## 7. 安全建议

- **不要**将 PAT 提交到公开 GitHub 仓库（使用环境变量管理）
- 定期轮换 Token（Coze 控制台可随时吊销重建）
- 可在 Coze 控制台为 Bot 设置每日调用上限，保护额度不被耗尽
